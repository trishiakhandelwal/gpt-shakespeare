GPT trained on Shakespeare text

Code and inspiration from Andrej Karpathy's https://github.com/karpathy/ng-video-lecture# 

Modifications/add-ons as of now: RMSNorm for normalization, RoPE for positional embeddings, KV cache to speed up generation by storing previously computed keys and values. 

Inspirations: "LLaMA: Open and Efficient Foundation Language Models" by Meta
